================================================================================
  SUPPORT & TROUBLESHOOTING
================================================================================

If pages don't load:
1. Check file paths are correct (case-sensitive on Linux servers)
2. Verify .html extensions are included in URLs
3. Check file permissions (should be 644 for files, 755 for folders)

If forms don't work:
1. Check browser console for JavaScript errors
2. Verify Supabase URL and anon key are correct
3. Check Supabase RLS policies allow public inserts

If images don't load:
1. Verify assets/ folder uploaded correctly
2. Check image paths in HTML (should be relative: ../assets/)
3. Check file permissions
