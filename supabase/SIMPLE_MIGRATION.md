# Supabase Migration Guide

This folder contains SQL migration files for setting up the TrueVow database schemas.

## Files in this folder:
- `migrations/001_phase1_website_schema.sql` - Marketing website database schema
- `migrations/001_initial_blog_schema.sql` - Blog content schema
- `EXPORT_TRUTHLINE_SCHEMA.sql` - Schema export script
- `AUTOMATED_MIGRATION_SCRIPT.sql` - Automated migration script

**Note:** These files need to be restored from backup or recreated from your Supabase dashboard.
