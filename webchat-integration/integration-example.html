TrueVow Sales WebChat Integration Example

Basic Integration

Add this code before the closing </body> tag:

<!-- TrueVow Sales WebChat Widget -->
<link rel="stylesheet" href="/webchat-integration/widget.css">
<script src="/webchat-integration/widget.js"></script>
<script>
  TrueVowSalesChat.init({
    apiBaseUrl: 'https://cs-support.truevow.com',
    position: 'bottom-right',
  });
</script>

Custom Position

TrueVowSalesChat.init({
  apiBaseUrl: 'https://cs-support.truevow.com',
  position: 'bottom-left', // or 'bottom-right'
});

How It Works

  1. Visitor clicks the chat button
  2. Enters email address
  3. System checks if email belongs to existing customer
  4. If customer: Shows redirect message to customer portal
  5. If prospect: Creates sales chat session
  6. Sales team receives chat in Sales inbox

Test Scenarios

  • New Prospect: Use a new email address → Should create sales chat
  • Existing Customer: Use an email from your customer database → Should show redirect
